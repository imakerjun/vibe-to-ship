# 스텝 2: DB + 기능 확장 + 배포

세션 1의 스텝 2~3과 동일한 흐름입니다. Lovable Cloud로 DB와 구글 로그인을 연결하고, 1차 배포까지 합니다.

## DB 연동 프롬프트 가이드

```
Lovable Cloud를 사용해서 데이터를 저장해줘.

저장할 데이터:
- {테이블명}: {컬럼 목록과 용도}

동작:
- 현재 하드코딩된 데이터를 DB에서 가져오도록 변경
- {사용자 액션}하면 DB에 저장
- 새로고침해도 데이터가 유지되어야 함
```

## 진행 순서

1. **DB 연동**: Lovable Cloud로 데이터 저장
2. **구글 로그인 추가**: 사용자 구분이 필요하면 추가
3. **추가 기능 요청 1~2건**: 핵심 기능 보완
4. **1차 배포**: Lovable Publish로 URL 생성
5. **동작 확인**: 배포된 URL에서 데이터 저장/유지 확인
