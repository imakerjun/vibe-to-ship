# Claude Code 설치 가이드

[Claude Code 공식 문서](https://docs.anthropic.com/en/docs/claude-code/setup)를 기반으로,
설치부터 첫 실행까지 스텝 바이 스텝으로 안내합니다.

## 시스템 요구사항

시작하기 전에 아래 조건을 확인하세요.

| 항목 | 조건 |
|------|------|
| **운영체제** | macOS 13.0 이상, Windows 10 이상, Ubuntu 20.04 이상 |
| **메모리** | 4GB RAM 이상 |
| **네트워크** | 인터넷 연결 필수 |
| **셸** | Bash, Zsh, PowerShell, 또는 CMD |
| **계정** | Claude Pro, Max, Teams, Enterprise 중 하나 (무료 플랜은 불가) |

<Callout type="warning">
**Windows 사용자**: Git for Windows가 반드시 설치되어 있어야 합니다.
[git-scm.com](https://git-scm.com)에서 설치할 수 있습니다.
</Callout>

## 설치 방법

<Steps>

### 방법 1: 공식 설치 스크립트 (권장)

자동 업데이트가 포함된 공식 설치 방법입니다.

**macOS / Linux**

터미널을 열고 아래 명령어를 붙여넣으세요.

```bash
curl -fsSL https://claude.ai/install.sh | bash
```

**Windows (PowerShell)**

PowerShell을 **관리자 권한으로** 열고 아래 명령어를 실행하세요.

```powershell
irm https://claude.ai/install.ps1 | iex
```

**Windows (CMD)**

명령 프롬프트를 열고 아래 명령어를 실행하세요.

```cmd
curl -fsSL https://claude.ai/install.cmd -o install.cmd && install.cmd && del install.cmd
```

<Callout type="info">
공식 설치 스크립트로 설치하면 **자동 업데이트**가 백그라운드에서 동작합니다.
별도로 업데이트를 신경 쓸 필요가 없습니다.
</Callout>

### 방법 2: Homebrew (macOS)

macOS에서 Homebrew를 사용하고 있다면 이 방법도 가능합니다.

```bash
brew install --cask claude-code
```

<Callout type="default">
Homebrew로 설치하면 자동 업데이트가 되지 않습니다.
업데이트하려면 `brew upgrade claude-code`를 직접 실행해야 합니다.
</Callout>

### 방법 3: npm (기존 방법)

Node.js 18 이상이 설치되어 있다면 npm으로도 설치할 수 있습니다.

```bash
npm install -g @anthropic-ai/claude-code
```

<Callout type="warning">
자동 업데이트가 필요하다면 **방법 1(공식 설치 스크립트)**을 권장합니다.
이미 npm으로 설치했다면, 방법 1로 다시 설치한 뒤 `npm uninstall -g @anthropic-ai/claude-code`로 기존 것을 제거할 수 있습니다.
</Callout>

</Steps>

## 설치 확인

설치가 끝나면 터미널에서 아래 명령어를 실행하여 정상 설치를 확인합니다.

```bash
claude --version
```

버전 번호가 출력되면 설치 완료입니다.

더 상세한 점검이 필요하면:

```bash
claude doctor
```

`claude doctor`는 설치 상태, 설정, 네트워크 연결 등을 종합적으로 진단합니다.

## 첫 실행 및 로그인

<Steps>

### 프로젝트 폴더로 이동

작업할 프로젝트가 있는 폴더로 이동합니다.

```bash
cd {프로젝트-폴더}
```

### Claude Code 실행

```bash
claude
```

### 로그인

처음 실행하면 브라우저가 열리며 로그인 화면이 나옵니다.
Claude 계정(Pro, Max, Teams, Enterprise)으로 로그인하세요.

<Callout type="info">
로그인은 최초 1회만 하면 됩니다.
이후에는 `claude` 명령어만 입력하면 바로 시작됩니다.
</Callout>

### 첫 대화 시작

로그인이 완료되면 바로 자연어로 대화할 수 있습니다.

```
이 프로젝트의 구조를 설명해줘.
```

</Steps>

## 자주 묻는 질문

### Q: 무료 Claude.ai 계정으로 사용할 수 있나요?

아니요. Claude Code는 **Pro, Max, Teams, Enterprise** 플랜에서만 사용할 수 있습니다.
이번 워크숍에서는 별도 제공되는 계정을 사용합니다.

### Q: VS Code에서도 쓸 수 있나요?

네. Claude Code는 터미널 CLI 외에도 **VS Code 확장**, **JetBrains 플러그인**, **데스크톱 앱**을 지원합니다.
이번 워크숍에서는 터미널 CLI를 사용합니다.

### Q: 인터넷이 끊기면 어떻게 되나요?

Claude Code는 인터넷 연결이 필수입니다.
오프라인에서는 동작하지 않습니다.

### Q: 설치가 안 될 때는?

| 증상 | 해결 |
|------|------|
| `command not found: claude` | 터미널을 닫고 새로 열어보세요. 셸 PATH가 갱신됩니다 |
| npm 권한 에러 | `sudo npm install -g @anthropic-ai/claude-code` 또는 공식 설치 스크립트 사용 |
| Windows에서 스크립트 실행 불가 | PowerShell 실행 정책 확인: `Set-ExecutionPolicy RemoteSigned` |
| 버전이 출력되지 않음 | `claude doctor`로 상세 진단 |
