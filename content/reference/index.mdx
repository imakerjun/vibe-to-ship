# 도구 가이드

## Lovable

AI와 대화로 웹앱을 만드는 도구입니다.

- 사이트: [lovable.dev](https://lovable.dev)
- 특징: 프롬프트 → 동작하는 웹앱, DB 연동(Lovable Cloud), 배포까지 원스톱

## Claude Code

터미널에서 AI와 대화하며 코드를 수정하는 도구입니다.

- 설치: `npm install -g @anthropic-ai/claude-code`
- 실행: 프로젝트 폴더에서 `claude` 입력
- 특징: 파일을 직접 읽고 수정, 로컬 환경에서 동작

## GitHub

코드를 저장하고 버전 관리하는 서비스입니다.

- 사이트: [github.com](https://github.com)
- 워크숍에서 사용: Lovable → GitHub 연결 → 로컬로 `git clone`
