# vibe-to-ship

<div className="workshop-narrative">

> **대화로 만들고 → DB를 연결하고 → 로그인을 붙이고 → 코드로 다듬고 → 배포해서 쓴다**

이 워크숍에서는 AI 바이브코딩 도구로 웹앱을 만들고,
데이터베이스와 인증을 연결하고, 코드를 개선하고, 배포하는 **전체 사이클**을 경험합니다.

세션 1에서 가이드를 따라 한 바퀴 돌고,
세션 2에서 같은 사이클로 **내 업무에 필요한 앱**을 직접 만듭니다.

</div>

## 오늘의 사이클

```
Lovable (프로토타입) → DB 연동 → 구글 로그인 → GitHub → Claude Code (개선) → 배포
```
