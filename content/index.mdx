# vibe-to-ship

<div className="workshop-narrative">

> **대화로 만들고 → DB를 연결하고 → 코드로 다듬고 → 배포해서 쓴다**

이 워크숍에서는 AI 바이브코딩 도구로 웹앱을 만들고,
데이터베이스를 연결하고, 코드를 개선하고, 배포하는 **전체 사이클**을 경험합니다.

오전에 가이드를 따라 한 바퀴 돌고,
오후에 같은 사이클로 **내 업무에 필요한 앱**을 직접 만듭니다.

</div>

## 오늘의 사이클

```
Lovable (프로토타입) → DB 연동 → GitHub → Claude Code (개선) → 배포
```

<Steps>

### 오전: 가이드를 따라 풀사이클 1회전

모두 같은 앱을 만들며 파이프라인 전체를 체험합니다.
문서를 따라하면 누구나 완주할 수 있습니다.

[오전 시작하기 →](/morning)

### 오후: 내 업무 앱 만들기

오전에 익힌 사이클을 자기 업무 과제에 적용합니다.
조별로 프로젝트를 정하고, 스프린트 3회로 완성합니다.

[오후 시작하기 →](/afternoon)

</Steps>
