# 스프린트 2: DB + 기능 확장 + 배포 (50분)

오전 Step 2와 동일한 흐름입니다. Lovable Cloud로 DB를 연결하고, 1차 배포까지 합니다.

## DB 연동 프롬프트 가이드

```
Lovable Cloud를 사용해서 데이터를 저장해줘.

1. {저장할 데이터} 테이블 생성
2. 현재 하드코딩된 데이터를 DB에서 가져오도록 변경
3. {사용자 액션}하면 DB에 저장되게 해줘
4. 새로고침해도 데이터가 유지되어야 해
```

## 진행 순서

1. DB 연동 (15분)
2. 추가 기능 요청 1~2건 (20분)
3. 1차 배포 — Lovable Publish (5분)
4. 동작 확인 (10분)
