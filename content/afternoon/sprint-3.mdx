# 스프린트 3: Claude Code + 배포 (40분)

오전 Step 3~4와 동일한 흐름입니다.

## 진행 순서

1. **GitHub 연결** (5분): Lovable에서 GitHub 연결
2. **로컬로 가져오기** (5분): `git clone` → `npm install`
3. **Claude Code로 개선** (20분): 기능 추가 또는 UI 개선
4. **배포** (10분): Lovable Publish 또는 `vercel deploy`

## Claude Code 요청 예시

```
이 프로젝트에서 개선할 점을 3가지 제안해줘.
그중 가장 효과적인 것 1개를 구현해줘.
```

```
이 앱의 UI를 좀 더 세련되게 만들어줘.
카드 디자인을 개선하고, 애니메이션을 추가해줘.
```

```
데이터를 CSV로 내보내기하는 기능을 추가해줘.
```
